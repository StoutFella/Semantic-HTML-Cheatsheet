address

Defines a contact information for the author/owner of a document or an article. The information can be an email address, URL, physical address, phone number, social media handle, etc. The text usually renders in italic.

Written by Jon Doe.
Visit us at: Box 564, Disneyland
USA

iframe

Use the iframe tag to embed another document within the current HTML document.

bdo

BDO stands for Bi-Directional Override.
It's used to override text direction, from left to right(ltr) or right to left(rtl).

This text will go right-to-left.

area

Defines an area inside an image map (an image map is an image with clickable areas). The area elements are always nested inside a map tag.

An image map, with clickable areas:

Note: The usemap attribute in img tag is associated with the map element's name attribute, and creates a relationship between the image and the map.

abbr

defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr."

The WHO was founded in 1948.

wbr

Specifies where in a text it would be ok to add a line-break. When a word is too long, the browser (different screen sizes) might break the word at the wrong place.

To learn AJAX, you must be familiar with the XMLHttpRequest Object.

The above highlighted word (XMLHttpRequest) breaks at 2 different places: Http and Request
